Witamy na stronie głównej

Jest to strona poświęcona nauce HTML oraz CSS. Kolejne lekcje i ćwiczenia będą w kolejnych stronach, podlinkowanych poniżej.

  • Przepis na hummus: nadawanie semantyki nieustruktyrzowanej stronie - należy zawsze używać semantycznie poprawnych elementów, jest to wykorzystywane potem w wielu miejscach, np.: przez wyszukiwarki internetowe (szukają po nagówkach h1), albo gdy ludzie z problemami wzroku mają program do czytania stron, wówczas wpierw czytane są nagłówki, aby nakreślić użytkownikowi co strona zawiera.

HTML i CSS są to technologie wykorzystywane przy tworzeniu stron internetowych, tzw. frontend. Są to podstawowe technologie, które każdy web developer musi znać. Poza tym jest jeszcze JavaScript, język programowania, działający w przeglądarce. Jest to język interpretowany, nie kompilowany, jak C# czy Java. W kazdej przeglądarce jest wbudowany silnik (napisany zazwyczaj w C), który zajmuje się wykonaniem kodu JS.

Najlepsze źródło informacji na temat technologii HTML, CSS oraz JS można znaleźć na stronach MDN (Mozilla Developers Network).
